This is the API

getModels

By passing in the appropriate options, you can search for available models

URL

https://virtserver.swaggerhub.com/flame/predict

HTTP Method

GET

Response Type

List

Parameters

  • Query: searchString String

    pass an optional search string for looking up inventory

  • Query: limit Integer

    maximum number of models to return

predict

URL

https://virtserver.swaggerhub.com/flame/predict/{modelID}

HTTP Method

POST

Response Type

List

Parameters

  • Path: modelID String

    model id to run prediction with

  • Form: upfile File

    The .sdf or mol file to upload.
